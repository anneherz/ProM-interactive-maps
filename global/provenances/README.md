# Interactive Map of the Present-Day Global Distribution of New Kingdom Monuments from the Memphite Region

## Overview

This deposit contains an interactive world map showing the present-day distribution of New Kingdom monuments from the Memphite region across museums and collections worldwide.

The map was designed and produced by **Anne Herzberg-Beiersdorf** using data from the **Prosopographia Memphitica (ProM)** database and implemented with **Leaflet.js**.

The deposit documents the workflow from the underlying relational database to the published visualization:

**ProM database → SQL query → data export → interactive map**

## Files

- `ProM_Weltkarte.html` — interactive world map
- `ProM_Weltkarte.png` — static preview image
- `world_map_query.sql` — SQL query used to extract the data underlying the map
- `world_map_data.csv` — exported dataset generated by the SQL query
- associated JavaScript, CSS, image, and data files, where applicable
- `README.md` — documentation of the deposit
